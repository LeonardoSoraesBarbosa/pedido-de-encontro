# Pedido de Encontro

Uma página interativa e divertida para convidar alguém especial para um encontro, com botões animados, calendário, confirmação por WhatsApp e atalho para Google Calendar.

## Rodar localmente

**Pré-requisito:** Node.js

1. Instale as dependências:
   `npm install`
2. Rode o projeto:
   `npm run dev`
3. Abra o endereço mostrado no terminal.

## Publicar no GitHub Pages

Este projeto já está configurado para publicar pelo GitHub Actions. Depois de enviar para o GitHub, ative o Pages em:

`Settings > Pages > Build and deployment > Source > GitHub Actions`
